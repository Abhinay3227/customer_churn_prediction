customer-churn-prediction
An end-to-end Customer Churn Prediction System built using Machine Learning and Streamlit. The application predicts customer churn, provides explainable insights using SHAP, and offers actionable business recommendations. Includes interactive dashboards, bulk prediction, and database integration for real-world usage.

📊 Customer Churn Prediction System
An end-to-end Machine Learning project that predicts whether a customer will churn or stay, along with explainable insights and business recommendations.

🔮 Customer Churn Prediction
🔬 SHAP-based Explainability
📊 Interactive Dashboard (EDA)
📁 Bulk Prediction (CSV Upload)
🗄️ Data Storage (Database)
💡 Business Action Suggestions
